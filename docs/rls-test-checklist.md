# RLS test checklist

- Anonymous users cannot select clinical records.
- Active staff can access patients granted through clinic membership.
- Deactivated staff cannot create or confirm clinical records.
- Patient clinic access controls wound, assessment, photo, and attachment visibility.
- Only the signer can create a clinical signature for themselves.
- Correction requests are visible to the requester and assigned consultant.
- Storage objects remain private and are accessed through signed URLs.
- No policy references billing or finance tables.
